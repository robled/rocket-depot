#### Rocket Depot
Rocket Depot is a simple graphical frontend for [rdesktop](http://www.rdesktop.org/) and [FreeRDP](http://www.freerdp.com/) with
support for connection profiles.  It runs on Linux desktops using GTK+ 3.
